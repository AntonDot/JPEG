# JPEG Парсер 

Простой инструмент на Python для анализа JPEG файлов.

## Возможности

- Просмотр заголовков JPEG файла (техническая информация)
- Конвертация изображений в ASCII арт для терминала

## Использование
python main.py image.jpg

### Что нужно

- Python 3.6+
- Библиотека Pillow

### Установка

```bash
pip install Pillow
